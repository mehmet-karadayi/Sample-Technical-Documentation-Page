Introduction

  Java is a class-based, Object-oriented programming language (OOP). It was designed to be a high-level, general-purpose language to allow developers to run Java code without the need for recompilation on all platforms that support Java. Java syntax is similar to that of C++ and C#. However, Java requires less low-level facilities than the previously mentioned languages.

  Java was first developed by James Gosling at Sun Microsystems in 1995 and different updated versions have been released ever since. The latest version, Java SE 15, was released on September 15, 2020. Java's portability as a language made it immensely popular amongst developers. The ability to run Java code on different machines with ony few or no changes to the code catapulted Java to become one most powerful programming language out there today.

Hello World

  One of the first programs that a beginner learning a new language is introduced to is called the Hello World program. The purpose of this program is to introduce some of the syntax of Java. The resulting output of this program is a simple message that prints "Hello, World!" to the screen. The program is written as follows:

public class Hello {
  public static void main(String[] args){
    //Generates the output to the screen
    System.out.println("Hello, World!");
  }
}

  When we compile and run this program the output we see on on the screen is:

Hello, World!

  Java programs are compromised of Classes and Methods. Methods are made up of series of Statements.

  •  A statement is a line of code that performs an operation. The example above uses a print statement to perform the task of outputting the phrase contained within the quotation (" ") to the display.

  •  A method is a series of statements which perform a task and are combined within a single named function. Our print statement is within the medthod called main in the example above.

      public static void main(String[] args)

  •  A class is a collection of methods. The class name for the program above is called Hello. Class names begin with capital letters whereas method names begin with lowercase letters. The class names must match the file name.

Variables and Operators

  In order to store values we need to declare variables. Variables are named locations where a given value is stored in. These values could be numbers, text, and any other types of data. Variables are declared using different types which determine what kind of values that variable can store. Here is an example of variable declaration:

char character;
String firstName;
int age;
boolean isTrue;

  There are four different types of variables that are declared in the above example:

  •  Characters are declared using the keyword char.
  •  String keyword is used to store "strings" of characters. Some type keywords are written with the first letter being uppercase. String is one of those types. If a variable name consists of two words, as in our example above, then the words are joined together and the first letter of the first word is written in lowercase. The following words start with an uppercase letter
  •  Integers are declared using the keyword int.
  •  Boolean values which are either true or false are declared using the keyword boolean.

  We can assign values to our variables by using an assignment statement. We can assign values during the declaration or after. Here are some examples of assignment statements:

String message = "Hello"; // Assigns the value "Hello" to the String variable named message
isTrue = true; // gives the boolean variable isTrue the value of true

  Operations can be performed on certain types of variables by using operators. Operators are symbols that represent simple computations. Here is a list of some of the operators that Java uses to perform computations:

  •  Addition => +
  •  Subtraction => -
  •  Multiplication => *
  •  Division => /
  •  Modulus => %
int x = 5;
int y = 6;
int sum = x + y;
System.out.println("The sum is: " + sum);

  The above example is a demonstration of the addition "+" operator. The two variable, x and y, are declared and assigned integer values. A third variable named sum is assigned the value of the sum of the values stored in the first two variables. We use the operator once again in our print statement, however, this time we are using it to combine our string text with the value stored in the sum variable. This is called string concatenation. The resulting output will be:

The sum is: 11

Conditionals and Logic

  In order to perform more complex computations we need to incorporate some logic into our programs. This complexity could be achieved through the use of relational operators. These operators are combined to form logic statements that enable us to performs tasks with when certain conditions are met. Here is a list of relational operators that are implemented by Java:

  •  x == y   // x is equal to y
  •  x != y   // x is not equal to y
  •  x > y   // x is greater then y
  •  x < y   // x is less than y
  •  x >= y   // x is greater than or equal to y
  •  x <= y   // x is less than or equal to y

  Java has three logical operators that are used to combine different relational operators. These operators are &&, ||, and !. && stands for the and, || for or, and ! stands for not. Lets look at an example of logical statements in action:

  • ( x > 4 ) && ( z != 0 )
  • ( w == 2 ) || ( y < 8 )
  • !( p == 5 )

  The first statement above is true if and only if both relational operators (those within the parenthesis) are true. The second example returns a value that is true if at least one of the relational operators is true. And the last example negates the truthy value of the relational operator. This means if the relational operator within the parenthesis is true then our final result is the opposite which is false and vice versa.

  We are able to build upon these basic operators to form conditional statements in Java. In order to create conditional blocks of code we use a combination of if, else if, and else statements. Here is an example of a simple block of code that represents a conditional statement:

if ( x == 5 ) {
  System.out.println(" x is equal to 5 ");
} else if ( x == 6 ) {
  System.out.println(" x is equal to 6 ");
} else {
  System.out.println(" x is equal to 0 ");
}

  We enter the conditional block by checking the first if statement. If the condition within the parenthesis is true then we run the code within the brackets and exit the conditional block without checking the other conditions. If not, then we move on to the else if statement. The same logic applies here as well. Finally if we haven't exited the conditional block before the last else statement then the we will exit by running the code within the else block.

Methods and Returns

  In order to have an organized program that has a consistent delegation of task we must implement different methods. Java has libraries of its own that come with their own classes and methods. One such example is the Math class. This class has methods that you could access and use within your code. When writing your own methods you must state the type of return value the method will give back to the caller. You could opt out of a return value by using the keyword void. Void methods will perform tasks without returning a value. If you would like the return value to be an integer then you must use the keyword int, for string string, and so on. You must also give each method a name. We follow the same rule as variable naming meaning if a method name is made up of more than one word then the first letter of the first word is lowercase. The following words start with capital letters. No spaces allowed. And finally, methods have parameters. The parameters are set when creating methods. You could opt out and leave the parameter section blank. However, if you want your methods to manipulate some data then you must name the parameters. Here is a code snippet that demonstrates a simple method structure:

public static int sum( int x, int y ){
  int sum = x + y;
  return sum;

  The example above is method named sum that has two parameter arguments that are both integer and it returns an integer value back to the caller. When the method is called, the values that are sent into the method are given to the parameter arguments. When we enter the method, the values stored in the parameter arguments are added and the total value is assigned to an integer called sum. Then the keyword return is used to return the value of sum.
